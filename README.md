# NoSQL-Social-Network-API

## Project Description
------
This challenge was to build an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This project use Express.js for routing, a MongoDB database, and the Mongoose ODM.
